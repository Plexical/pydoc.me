pymod.me - quick redirects to Python documentation

pymod.me is a shortcut service that will redirect you to official Python documentation by opinionated jumps using simple search terms in the URL. I have been using mdn.io for a while and wanted to have the same funcitonallity when I was developing Python too. Try for example:

  • /itertools - Jumps to the official Itertools docs.
  • /chain.from_iterable - Jumps to the official itertools.chain.from_iterable docs.

pymod.me is opinionated in the sense that it will currently default to Python 3 documentation. If you want to jump directly to Python 2 docs, put a 

/2/
before your search term, like this:
  • /2/itertools - Jumps to the official Itertools for Python 2.x docs.
  • /2/chain.from_iterable - Jumps to the official itertools.chain.from_iterable for Python 2.x docs.
